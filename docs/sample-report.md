# Sample Research Report

## Executive Summary

This sample shows the expected shape of a finished report: a decision-oriented summary, cited findings, a comparison section, and recommendations.

## Key Findings

### Search Quality

- Tavily-backed search provides richer source material than snippet-only DuckDuckGo fallback.
- The analysis phase is expected to compare named options instead of summarising search results.

### Report Structure

- Reports use markdown headings, bullets, and comparison tables.
- The web viewer supports GFM tables and scrollable wide content.

## Comparison

| Option | Strength | Trade-off | Best for |
|--------|----------|-----------|----------|
| Tavily Search | Rich search results | Requires API key | Production research |
| DuckDuckGo fallback | No key required | Lower quality snippets | Local testing |

## Recommendations

- Use Tavily for serious comparison topics.
- Keep DuckDuckGo as a free fallback for demos and development.
